### TicTacToe ###

===========================================================================
DESCRIPTION:

TicTacToe shows how to customize different parts of your application for iOS 7. 
It shows how to use and customize tintColor, renderingMode, and how to customize standard UIKit controls.
It also shows how to create your own custom controls.
For more information, see Session 214 from WWDC 2013.

===========================================================================
BUILD REQUIREMENTS:

iOS 7.0 or later

===========================================================================
RUNTIME REQUIREMENTS:

iOS 7.0 or later

===========================================================================
PACKAGING LIST:

- TTTPlayViewController, TTTMessagesViewController, and TTTProfileViewController are the application's main view controllers
- TTTRatingControl, TTTGameView, and and TTTCountView show how to make custom controls
