### FlightFinder ###

===========================================================================
DESCRIPTION:

This is a sample used as a demo of a simple multi-process application in the WWDC 2012 session "Cocoa Interprocess Communication with XPC." It shows basic setup of an NSXPCConnection, NSXPCListener, and NSXPCInterface. It also shows a simple model object that implements NSSecureCoding.
