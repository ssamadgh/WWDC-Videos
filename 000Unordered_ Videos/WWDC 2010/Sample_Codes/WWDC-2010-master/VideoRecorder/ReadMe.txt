MyImagePicker
======================

Demonstrates how to create a custom UI for the camera variant of the UIImagePickerController and how to programmatically control video recording.


Build Requirements
------------------
iPhone SDK 4.0 or later


Runtime Requirements
--------------------
iPhone OS 4.0 or later


Using the Sample
----------------
Open the project, build it and run it on an iPhone 4.


Packaging List
--------------



VideoRecorderAppDelegate.{h,m}
- Default application delegate created by Xcode.

RootViewController.{h,m}
- Main view controller for the VideoRecorder app. It is where all the logic to control the custom video recording user interface is located.
