### GameConsole ###

===========================================================================
DESCRIPTION:

Game Console uses KVO and NSDictionary to implement a console. The techniques demonstrated in this sample can be used to  enable rapid iteration of UI changes and gameplay tweaks during development.

===========================================================================
BUILD REQUIREMENTS:

iPhone SDK 4.0

===========================================================================
RUNTIME REQUIREMENTS:

iPhone OS 3.2
